# 🎬 Movie Explorer WebApp

## 📌 Project Overview
This is a simple web application that allows users to search and explore movies using real-time data from the OMDb API. The application is built using HTML, CSS, and JavaScript.

## 🎯 Purpose
The purpose of this project is to practice working with APIs, handling user input, and building dynamic web interfaces with search, filtering, and sorting features.

## 🌐 API Used
- OMDb API
- https://www.omdbapi.com/

## ✨ Features (Planned)
- 🔍 Search movies by title
- 🎭 Filter by type (movie, series, episode)
- 📅 Filter by release year
- 📄 View movie details (poster, plot, rating)
- ⭐ Sort results (planned for future)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)
- Fetch API

## ⚙️ Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/dipu01_code/movie-explorer.git
```

2. Open the project folder:
```bash
cd movie-explorer
```

3. Open `index.html` in your browser

## 🔑 API Setup
- Get your free API key from OMDb API
- Replace `YOUR_API_KEY` inside `script.js`
